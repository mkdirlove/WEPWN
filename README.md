# WEPWN
A simple attack tool for WEP encrypted networks.
![WEPwn](https://github.com/mkdirlove/WEPWN/blob/main/wepwn.png)
#
# INSTALLATION & USAGE
    
    $ git clone https://github.com/mkdirlove/WEPWN
    
    $ cd WEPWN
    
    $ sudo bash install.sh
    
    $ sudo wepwn.sh
